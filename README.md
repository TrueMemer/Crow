#Crow
=============================

Discord library written in C.

Tested on Ubuntu 16.04.

##Installation

Requirements

- gcc, make
- [json-c](https://github.com/json-c/json-c)
- libcurl
- [libwsclient](https://github.com/payden/libwsclient)

##Execute the following commands:
```
    $ ./autogen.sh
    $ ./configure
    $ make
    $ sudo make install
```

To run bot execute ```./bin/crow```.
